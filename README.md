# 🎓 Student Dashboard (HTML + CSS)

A modern and responsive **Student Dashboard** built using **pure HTML and CSS**.  
This project demonstrates clean UI design, responsive layouts, and multi-page navigation without using JavaScript.

---

## 📌 Features

- 📊 Dashboard overview (subjects, assignments, attendance)
- 📚 Subjects page with progress bars and grades
- 📝 Assignments page with status indicators
- 📊 Results page with marks and grades
- 🌙 Dark mode (CSS-only toggle)
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox + Grid)  
- CSS Variables  


---

## 📱 Responsive Design

Built using **CSS Grid and Flexbox**, optimized for:

- Desktop 💻  
- Tablet 📱  
- Mobile 📲
